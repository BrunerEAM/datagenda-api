# Datagenda API

Backend da plataforma **Datagenda**, um sistema SaaS desenvolvido para gerenciamento de agendamentos, serviços, produtos e clientes para empresas de prestação de serviços.

A API fornece toda a estrutura necessária para o funcionamento da aplicação web, incluindo autenticação, gerenciamento de dados e integração com o frontend.

---

##  Tecnologias Utilizadas

- Node.js
- Express.js
- PostgreSQL
- JWT (JSON Web Token)
- Multer
- Knex.js
- REST API

---

##  Funcionalidades

A API oferece suporte para:

- Autenticação de usuários com JWT
- Cadastro e gerenciamento de empresas
- Gestão de clientes
- Cadastro de produtos
- Cadastro de serviços
- Controle de estoque
- Gerenciamento de pacotes e combos
- Upload de imagens
- Histórico de alterações
- Paginação e filtros em consultas
- Estrutura preparada para multiempresa

---

##  Arquitetura

A aplicação segue o padrão de arquitetura baseado em **API REST**.
